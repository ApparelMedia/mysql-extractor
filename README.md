# MySQL Extractor

This node package is to easily extract the MySQL Schema to a json format which then can be generated into various migration files.

## Usage
```javascript

```

## Detects the following types
- boolean
- date
- dateTime
- decimal
- geometry
- integer
- string (VarChar)
- text
- bigInteger 
- char
- double
- enum
- float
- json
- jsonb
- longText
- mediumInteger
- mediumText
- smallInteger
- time
- tinyInteger
- timestamp
